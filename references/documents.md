## 参考文献格式

### 引用规范
- **学术论文**: 作者, 标题, 期刊, 年份
- **在线文档**: 组织/作者, 标题, URL, 访问日期
- **书籍**: 作者, 书名, 出版社, 年份
- **代码库**: 作者, 项目名称, GitHub URL, 版本

### 参考模板
```bibtex
@article{example2024,
  author = {Author Name},
  title = {Article Title},
  journal = {Journal Name},
  year = {2024},
  volume = {1},
  pages = {1--10}
}

@book{book2024,
  author = {Author Name},
  title = {Book Title},
  publisher = {Publisher Name},
  year = {2024}
}
```
