这是一个涵盖技术、投资、运动和生活等多个领域的个人知识管理系统。

## 📚 目录结构

```
├── personal_wiki.md              # 主导航和框架
├── tech/                         # 技术/AI领域
│   ├── local_models.md          # 本地模型部署与优化
│   ├── openclaw.md              # OpenClaw框架文档
│   ├── dingtalk_integration.md  # 钉钉生态集成指南
│   └── ai_trends.md             # AI技术趋势追踪
├── finance/                      # 投资金融领域
│   ├── investment_theory.md     # 投资理论与策略
│   ├── risk_management.md       # 风险管理指南
│   ├── market_analysis.md       # 市场分析方法
│   └── investment_psychology.md # 投资心理与决策
├── health/                       # 运动健康领域
│   ├── running_training.md      # 跑步训练计划
│   ├── heart_rate_analysis.md   # 心率数据分析
│   ├── fitness_nutrition.md     # 健身营养知识
│   └── equipment_reviews.md     # 运动装备评测
└── README.md                     # 项目说明
```
